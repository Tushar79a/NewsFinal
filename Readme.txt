Overview
Hacker News website is displaying news from all over the world where user can see the comments on it and also can perform various activities on it, for example they can like the news, or they can hide the news if they don't want to see any specific news. Also they can see real time Id and upvote line chart for any speific page.  

Technical overview
- The Hacker News web site is using Angular 8 at the front end  and https://hn.algolia.com/api for fetching and pagination of real data from server.
- The web site is hosted by Heroku.
